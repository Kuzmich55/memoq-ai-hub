## What's New

- Reworked the Dashboard into a clear five-step setup journey: Integration, AI Services, optional Assets, Setup, and Translation Records.
- Added responsive application navigation for wide, compact, and narrow windows. At 768 px and below, navigation now opens in a drawer instead of reserving a permanent side rail.
- Added consistent page titles, descriptions, and navigation groups so each workspace is easier to identify and scan.
- Reduced information density across AI Services, Setup, Assets, and Translation Records with staged sections, focused filters, and progressive disclosure for advanced or diagnostic controls.
- Added Save, Discard, and Stay protection when leaving an edited AI service or translation profile, plus a visible unsaved state and protection when closing the window.
- Restored the last valid page, navigation preference, and per-page scroll position between sessions.
- Improved keyboard access and focus feedback for AI service and profile selection, History actions, and Assets controls.
- Added behavior-level renderer coverage for responsive navigation, state restoration, dirty-navigation protection, and keyboard activation.

## Notes

- Existing AI service connections, translation profiles, uploaded assets, and translation records are preserved.
- This release does not change the memoQ plugin contract, Provider API configuration format, or local storage contracts.
- Data tables continue to scroll inside their own regions on narrow windows so the application page does not overflow horizontally.
- The interface and user guides remain available in English and Simplified Chinese.
