## What's New

- Reduced median desktop runtime startup time by 57.59% through lazy loading of provider SDKs, spreadsheet/XML tooling, and feature pages.
- Reduced initialized runtime RSS growth by 49.51% while preserving existing provider, asset, history, and translation behavior.
- Added a compact portable 7z package that is 33.28% smaller than the previous portable ZIP baseline, while retaining the ZIP for compatibility.
- Reduced packaged runtime dependencies, source files, type declarations, source maps, and unused Electron locales without changing the shipped runtime contract.
- Added repository-owned startup, memory, package-size, lazy-loading, packaged-dependency, and release-asset regression checks.

## Notes

- Existing AI service connections, translation profiles, uploaded assets, translation records, and local storage formats are preserved.
- This release does not change the memoQ plugin contract, Provider API configuration format, gateway routes, or update manifest compatibility.
- The existing ZIP remains available; the 7z is an optional smaller download containing the same complete portable application files.
- Performance figures use seven isolated Windows Node.js 22 samples on the same machine and compare against the v1.0.22 baseline.
