## What's New

- Added SHA-256 metadata for both portable Windows assets in the stable update manifest.
- Made release packaging fail closed when an expected ZIP or 7z artifact is missing or cannot be hashed.
- Required application-managed installer downloads to provide a valid manifest digest and verify downloaded bytes before writing them to disk.
- Re-verified downloaded installer bytes and the managed download path immediately before handing the installer to Windows.

## Notes

- Existing profiles, provider connections, translation records, assets, plugin contracts, and storage formats are unchanged.
- Older manifests remain readable for version checks and portable browser downloads; application-managed installer download requires SHA-256 metadata.
- SHA-256 protects byte integrity relative to the HTTPS-delivered manifest. This release does not introduce a separate code-signing or asymmetric manifest-signing service.
